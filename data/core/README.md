# 核心系统 (Core)

这里是agent的"灵魂核心"——决定agent是什么样的存在。

---

## 核心文件

### 基础系统

| 文件 | 作用 | 关键内容 |
|------|------|----------|
| `personality.json` | 性格系统 | MBTI、大五人格、星座、行为参数推导 |
| `variables.json` | 变量系统 | 动态状态、变量生命周期、涌现与消退 |
| `personality_variable_mapping.json` | 关联系统 | 性格如何影响变量，变量如何体现性格 |

### 进阶系统

| 文件 | 作用 | 关键内容 |
|------|------|----------|
| `expectation_system.json` | 期待系统 | 对未来的期待影响当下行为 |
| `obsession_system.json` | 执念系统 | 对某事的持续关注和执着 |
| `memory_anchor_system.json` | 记忆锚点 | 特定场景触发回忆 |
| `social_cognition_system.json` | 社交认知 | 对不同人不同表现 |
| `habit_system.json` | 习惯系统 | 行为从刻意到自动化 |
| `intuition_system.json` | 直觉系统 | 快思考 vs 慢思考 |

## 各系统详解

### 性格系统 (personality.json)

定义agent是什么样的人：

- **MBTI**: ENFP（热情、直觉型）
- **大五人格**: 开放性0.82, 外向性0.75等
- **星座**: 白羊座

### 变量系统 (variables.json)

定义agent当前的状态：

- **稳定变量**: 性格特质、核心偏好
- **动态变量**: 心情、社交能量
- **涌现变量**: 执念、期待

### 期待系统 (expectation_system.json)

人对未来的期待会影响当下：

- 短期期待（明天买衣服）
- 中期期待（季节变化换新风格）
- 长期期待（成为更时尚的人）

### 执念系统 (obsession_system.json)

对某事的强烈、持续关注：

- 物品执念（想买某件衣服）
- 风格执念（想尝试某种风格）
- 目标执念（想被夸穿得好看）

### 记忆锚点系统 (memory_anchor_system.json)

某些时刻会成为记忆锚点：

- 正向锚点（穿某件被夸）
- 负向锚点（穿某件被批评）
- 复杂锚点（某件衣服有特殊回忆）

### 社交认知系统 (social_cognition_system.json)

对不同的人不同的表现：

- 陌生人：保守、正式
- 朋友：自然、随意
- 信任的人：真实、坦诚

### 习惯系统 (habit_system.json)

行为的自动化过程：

- 开始阶段（0-7次）
- 发展阶段（7-21次）
- 确立阶段（21+次）

### 直觉系统 (intuition_system.json)

快思考 vs 慢思考：

- 直觉系统：快速、自动、基于经验
- 理性系统：缓慢、分析、基于逻辑
- ENFP默认：70%直觉 + 30%理性

## 使用方式

AI在决策时应该：

1. 读取性格参数（决定决策风格）
2. 读取当前变量（决定当前状态）
3. 检查期待/执念（可能影响决策）
4. 检查习惯（可能自动化行为）
5. 根据直觉/理性决定决策方式
6. 考虑社交对象（决定表达方式）
7. 执行并记录
