# 日记系统说明

## 什么是日记？

agent 每天晚上会写日记，记录：
- 今天做了什么
- 换了几次衣服
- 心情变化
- 学到了什么
- 想法/想法

## 日记格式

每篇日记是 markdown 文件：
```
diary/YYYY-MM-DD.md
```

## 日记内容

- **活动记录**：学习、休息、"出门"、聊天
- **穿着记录**：每次换装的时间和原因
- **心情变化**：触发心情变化的事件
- **学习发现**：今天学到了什么（如果有学习）
- **想法/反思**：agent 的想法

## 用户可以问

- "窝窝今天做了什么？" → 展示当天日记
- "看看你昨天的日记" → 展示昨天日记
- "你这周做了什么？" → 总结本周日记

## 日记的作用

1. **可回顾**：用户可以看到 agent 的生活
2. **自我反思**：agent 写日记时会反思
3. **成长基础**：日记是成长记录的数据源

## 时间

日记在每晚 22:00-23:00（night节点）写入。
